# DuoSbcGo 📘

每日 3 個英文生字 + **間距複習**（spaced repetition），離線用得到嘅手機 apps。
專為香港補習學生設計：主動回憶測驗（睇中文打英文），禁用 paste 杜絕 copy-paste。

> Phase 1：純本地版（每部手機自己記進度）。
> Phase 2（之後）：加 Firebase 同步 + 老師後台，一覽睇晒全班進度。

---

## 點樣喺自己電腦試（唔使 setup）

呢個 apps 係純靜態網頁，但要經 http 服務先可以用到 service worker（離線 / 安裝）。

**方法 A — Claude Code 已經幫你起咗個本地 server：**
直接喺手機 / 電腦瀏覽器打開條 link 就得（我會俾你）。

**方法 B — 自己起（如果有 Python）：**
```bash
cd DuoSbcGo
python -m http.server 8000
# 然後瀏覽器打開 http://localhost:8000
```

**方法 C — 最快（但冇離線安裝）：**
雙擊 `index.html` 直接用瀏覽器打開。功能用到，但「加至主畫面」同離線功能會有限制。

## 點樣安裝去手機（PWA）

1. 用手機瀏覽器打開個網址。
2. **iPhone / iPad（Safari）**：下面分享按鈕 →「加至主畫面」。
3. **Android（Chrome）**：右上選單 →「新增至主畫面 / 安裝應用程式」。
4. 之後好似原生 apps 咁有個 icon，離線都開得到。

## 點樣加 / 改字

打開 `words.js`，照格式加新一行就得：
```js
{ en: "example", pos: "名詞", zh: "例子" },
```
- `en`：英文（小寫、基本形）
- `pos`：詞性（中文：名詞 / 動詞 / 形容詞 / 副詞 / 介詞 / 連詞 / 代詞 / 助動詞…）
- `zh`：中文意思

改完之後重新發佈，學生下次打開就自動有新字。
> Phase 1 改字係直接改呢個檔（或講俾 Claude 聽，佢幫你改）；Phase 2 會有後台界面。

## 間距複習係點運作

每個字有一個「box」（等級），答啱升一級，間距越來越長：

| Box | 下次複習 |
|----|--------|
| 0 | 1 日後 |
| 1 | 3 日後 |
| 2 | 7 日後 |
| 3 | 14 日後 |
| 4 | 30 日後 |
| 5 | 60 日後（畢業） |

答錯就打返 box 0，聽日再試。**字永遠唔會被掉低** —— 解決「學完一個禮拜就搞亂 / 唔記得」嘅問題。

## 除錯

主頁底部「⚙️ 設定 → 🛠 除錯」：
- **⏩ 快進 1 日**：模擬聽日，用嚟驗證間距排程（唔使真係等）。
- **🗑 重置全部**：清晒所有學習紀錄，由頭嚟過。

## 老師後台 + 多學生（Sprint B，需 Firebase）

學生進度自動同步上雲端，老師一個畫面睇晒全班。

**第一次設定（老師做一次）：**
1. 用瀏覽器打開 **`teacher.html`**（例如 http://localhost:8000/teacher.html ）。
2. 設定**班級碼**（例如 `3A`）同**後台密碼** → 建立。
3. 之後每次入後台都要入密碼。

**學生加入：**
- 學生打開主網址 → 填**班級碼** + **名字** → 加入。
- 之後每次開自動記住身份，進度離線照記、有網自動同步。

**老師見到：** 每個學生嘅 streak、最後活躍、學咗幾多字、整體正確率、**最弱嘅 3 個字**，實時更新。

> 需要免費 Firebase project（見 `firebase-config.js`）。目前用 Firestore test mode；正式安全鎖定可之後加。

## 功能一覽

- **學習流程**：睇 3 個字 → 選詞性（多選）→ 選中文 → 打字（主動回憶）
- **間距複習**：1 / 3 / 7 / 14 / 30 / 60 日，字永遠唔會被掉低
- **neediness 排序**：錯得多 / 過期耐嘅字優先複習
- **8 分鐘預算**：每節複習有上限（預設 12），到期太多會自動減少新字，保住每節時長
- **錯誤統計**：每字記 attempts / 正確率 / 連續錯
- **錯字簿** 📚：正確率低（<70%）嘅字集中操練
- **英文發音** 🔊：Web Speech API，免費離線
- **防作弊**：測驗時唔顯示英文字 + 禁 paste
- **設定可調**：揭曎秒數 / 鎖定秒數 / 複習上限 / 每日新字數
